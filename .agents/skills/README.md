# Ready2Agent Learn Skills

Le skills in `Ready2Agent Learn` sono esempi didattici riutilizzabili.
Non sono una governance rigida enterprise.

## Obiettivo

- spiegare come dare istruzioni stabili a un coding agent
- mantenere il repository semplice, adattabile e leggibile
- supportare un modello skills-first senza appesantire il flusso

## Regole base

- le skills possono essere modificate, eliminate, unite o accorpate
- non devono essere troppe
- ogni skill deve restare breve, chiara e utile
- non copiare automaticamente tutte le skill del repository operativo

## Skill principale

- `ready2agent-learn-core`: skill di default per il lavoro su questo repository

## Skill opzionali

In questo repository sono facoltative. Aggiungile solo se servono davvero a un task didattico specifico.

## Adapter Learn

Usa `READY2AGENT_LEARN_ADAPTER.md` per tradurre regole del framework operativo nel contesto educativo `learn`.

## Messaggio chiave

Le skills in `Ready2Agent Learn` sono un modello apprendibile e adattabile: insegnano come costruire istruzioni riutilizzabili per agenti di coding, senza bloccare l'utente in una struttura proprietaria pesante.
