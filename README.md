# Ready2Agent Learn

Laboratorio open-source per imparare a costruire applicazioni Next.js AI-native con agenti di coding.

## Disclaimer

Questa non e la versione enterprise completa di Ready2Agent.
Questa e la versione Learn, pubblica, divulgativa e formativa.

## Cosa contiene

- landing Next.js pubblica orientata al metodo
- documentazione metodo e governance
- prompt agent-native sequenziali
- workflow didattico tracciato
- caso studio Auth/IAM guidato
- skeleton architetturale frontend/server/shared

## Cosa non contiene

- implementazione enterprise completa
- dashboard privata pronta
- moduli business reali
- codice cliente
- sistema Auth/IAM gia completo

## Come iniziare

```bash
npm install
npm run dev
```

Poi:

1. apri la landing su `http://localhost:3000`
2. leggi `documentation/quickstart/human_start_here.md`
3. per agenti leggi `documentation/quickstart/agent_start_here.md`
4. apri `workflow/requirements/r2a-learn-auth-iam-case-study/00-master-requirement.md`
5. usa `prompts/00-agent-onboarding.md`

## Percorso consigliato

- Umano: README -> quickstart umano -> metodo -> caso studio -> workflow -> prompt
- Agente: README -> full AI context -> requisito master -> prompt corrente -> task incrementale
