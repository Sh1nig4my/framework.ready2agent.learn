# Documentation Hub

`documentation/` e il centro operativo di Ready2Agent Learn.

## Come orientarsi

- `quickstart/`: ingressi rapidi per umano e agente
- `method/`: principi operativi agent-native
- `case-study/`: caso studio Auth/IAM in formato didattico
- `project-meta/`: governance e separazione repository
- `R2A_LEARN_full-ai-context.md`: contesto completo per coding agent

## Uso con workflow e prompt

1. leggi requisito master
2. analizza scope e confini
3. usa il prompt corretto per fase
4. implementa un task alla volta
5. valida con quality gate
6. riallinea documentazione
