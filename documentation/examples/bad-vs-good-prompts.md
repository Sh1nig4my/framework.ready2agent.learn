# Bad vs Good Prompts
## Esempio debole
"Fai una dashboard migliore possibile."
## Perche non funziona
Obiettivo ambiguo, nessun vincolo, output non verificabile.
## Esempio Ready2Agent Learn
Prompt con ruolo, task unico, file target, vincoli, output atteso, quality check.
## Perche funziona
Riduce drift e produce risultato valutabile.
## Checklist riutilizzabile
- task unico
- vincoli espliciti
- output verificabile
- quality gate richiesti
