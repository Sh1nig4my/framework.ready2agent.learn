# Bad vs Good Task Breakdown
## Esempio debole
"Implementa tutto il modulo auth in un task."
## Perche non funziona
Task monolitico, review difficile, rischio regressioni alto.
## Esempio Ready2Agent Learn
Task incrementali con dipendenze e DoD per ogni passo.
## Perche funziona
Permette controllo qualità continuo.
## Checklist riutilizzabile
- 1 obiettivo per task
- dipendenze esplicite
- DoD verificabile
- ordine implementativo chiaro
