# First Mission

## Obiettivo

La First Mission e la golden path di Ready2Agent Learn: in 30-60 minuti trasformi un'idea confusa in un output verificabile chiamato `First Mission Dossier`.

Se non sai da dove iniziare, inizia da qui.

## Cosa fai in questa missione

1. chiarisci l'idea iniziale
2. definisci requirement e confini
3. produci un prompt agent-ready
4. scomponi il lavoro in task piccoli
5. definisci checklist di verifica
6. chiudi con retrospettiva e next step

## Output richiesto

Output finale obbligatorio:

`First Mission Dossier`

Usa questo template:

- `documentation/first-mission/first-mission-dossier-template.md`

## Regole della missione

- niente runtime enterprise
- niente database reale
- niente auth production-ready
- niente feature extra fuori scope
- ogni sezione deve essere verificabile

## Collegamenti rapidi

- Wiki missione: `/wiki/first-mission`
- Prompt onboarding: `prompts/00-agent-onboarding.md`
- Hub prompts: `prompts/README.md`
- Hub exercises: `exercises/README.md`
- Hub playground: `playground/README.md`

## Criterio di completamento

La missione e completata quando il dossier contiene sezioni compilate, confini espliciti, checklist verificabile e next step coerente.
