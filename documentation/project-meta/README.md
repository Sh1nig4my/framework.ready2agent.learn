# Project Meta

La cartella `documentation/project-meta/` contiene i documenti di governance del repository pubblico Learn:

- confine pubblico/privato
- definizione ufficiale della release v1.0.0
- regole per contributor e coding agent
- guida editoriale unica

Leggila prima di aprire PR che toccano struttura, scope o posizionamento del progetto.

Guida consigliata per uniformare il copy: `documentation/project-meta/editorial-style-guide.md`.
