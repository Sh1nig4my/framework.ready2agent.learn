# Release Definition - Learn v1.0.0

## Cosa significa v1.0.0

La v1.0.0 di Ready2Agent Learn segna maturita didattica pubblica, non maturita di runtime backend enterprise.

## Criteri di maturita didattica

- README come root experience operativa
- landing locale coerente e navigabile
- playground pronto all'uso
- esercizi progressivi livelli 1-4
- tracker esercizi e tracker case study
- teaching guide per aula/workshop
- quality gate tecnici e strutturali
- documentazione metodo + esempi bad/good

## Vincolo architetturale della v1

- nessun backend reale obbligatorio
- nessun database reale
- nessun modulo enterprise privato

## Consegna minima di release

- struttura repository chiara
- prompt 00-06 allineati al metodo
- boundary pubblico/privato esplicito
- script `check:structure` funzionante
- workflow CI quality eseguito su push/PR

## Perimetro prodotto

Learn e un laboratorio didattico open-source dove impari a comandare processo e qualità con coding agent.
