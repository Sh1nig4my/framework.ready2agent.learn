# Context to Prompt

## Livello

1 - Foundations

## Obiettivo

Scrivere un prompt operativo forte a partire dal contesto.

## Perche e importante

Prompt debole produce output non governabile.

## Scenario

Hai contesto completo ma prompt troppo generico.

## Input disponibili

- context pack
- requirement

## Task richiesto

Genera prompt con ruolo, task, input, output, vincoli, controlli.

## Vincoli

- niente linguaggio vago
- includi formato output finale

## Prompt consigliato

`prompts/04-implementation-start.md`

## Output atteso

Prompt incollabile in Codex/OpenCode.

## Criteri di valutazione

- obiettivo e misurabile
- controlli verificabili
- confini chiari

## Errori comuni

- task multipli nello stesso prompt
- output atteso non specificato

## Aggiorna il tracker

Aggiungi prompt usato e note.

## Variante avanzata

Scrivi anche una versione "strict mode" con vincoli piu duri.
