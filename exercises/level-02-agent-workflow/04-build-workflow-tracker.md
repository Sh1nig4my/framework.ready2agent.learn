# Build Workflow Tracker
## Livello
2 - Agent Workflow
## Obiettivo
Costruire un tracker operativo per seguire task e decisioni.
## Perche e importante
Senza tracker perdi contesto tra sessioni.
## Scenario
Team con piu iterazioni e output disallineati.
## Input disponibili
Backlog task e requirement.
## Task richiesto
Definisci template tracker con stato, rischi, quality gate, next step.
## Vincoli
Formato semplice e aggiornabile.
## Prompt consigliato
`prompts/workflow-maintenance/03-update-workflow-tracker.md`
## Output atteso
Tracker riusabile per progetto.
## Criteri di valutazione
Leggibilita e tracciabilita.
## Errori comuni
Tracker troppo verboso.
## Aggiorna il tracker
Compila almeno 3 righe reali.
## Variante avanzata
Aggiungi colonna ownership.

## Checklist di completamento

- [ ] tracker con stato, rischi, gate e next step
- [ ] almeno 3 righe compilate
- [ ] formato leggibile e aggiornabile
- [ ] scorecard compilata
