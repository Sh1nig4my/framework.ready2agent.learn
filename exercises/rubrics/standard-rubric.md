# Standard Rubric (0-18)

Usa questa rubrica per valutare in modo coerente gli esercizi Ready2Agent Learn.

## Dimensioni (6)

| Dimensione | Cosa valuta |
|---|---|
| Requirement clarity | Obiettivo chiaro e verificabile |
| Scope discipline | In-scope/out-of-scope espliciti |
| Prompt quality | Ruolo, contesto, input, output, controlli |
| Agent control | Capacita di correggere drift e assunzioni |
| Validation | Output verificabile con gate/checklist |
| Documentation trace | Decisioni, rischi e next step tracciati |

## Scala per ogni dimensione

- `0`: assente
- `1`: debole
- `2`: buono
- `3`: eccellente

## Totale e interpretazione

- `0-7`: insufficiente
- `8-12`: base
- `13-16`: buono
- `17-18`: eccellente

## Regola didattica

Se non e verificabile, non e finito.
