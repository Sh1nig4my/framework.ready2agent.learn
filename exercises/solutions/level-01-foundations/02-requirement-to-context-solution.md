# Soluzione esempio - Requirement to Context

## Context pack sintetico

- Obiettivo task: generare shell dashboard didattica.
- File coinvolti: `src/app/page.tsx`, `src/app/globals.css`, `documentation/README.md`.
- Vincoli: no backend reale, no dipendenze extra, responsive.
- Quality gate: lint, build, typecheck.
- Non-obiettivi: auth reale, persistenza dati.

## Note correzione

Controlla che il contesto non sia solo narrativo ma operativo.
