# Workflow Maintenance 02 - Generate Workflow Tasks

## A cosa serve

Trasforma capitoli workflow in task piccoli, eseguibili e verificabili.

## File da leggere

- capitoli workflow
- requirement master

## Prompt pronto da copiare

```text
Converti i capitoli workflow in task operativi.
Per ogni task includi: descrizione, dipendenze, definizione di completamento, quality gate.
Mantieni task incrementali e senza scope creep.
```

## Output atteso

- backlog task ordinato
- dipendenze esplicite

## Checklist umana

- ogni task e piccolo
- ogni task ha completion criteria

## Errori comuni

- task troppo grandi
- task senza criteri verificabili
