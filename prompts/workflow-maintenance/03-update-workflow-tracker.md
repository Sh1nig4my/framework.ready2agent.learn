# Workflow Maintenance 03 - Update Workflow Tracker

## A cosa serve

Aggiorna avanzamento, blocchi, decisioni e prossimi step in modo tracciabile.

## File da leggere

- task completati/in corso
- output quality gate
- documentazione aggiornata

## Prompt pronto da copiare

```text
Aggiorna il tracker workflow con stato corrente.
Includi: task completati, task bloccati, decisioni prese, rischi residui, next steps.
Non inventare avanzamenti non verificati.
```

## Output atteso

- tracker coerente con stato reale

## Checklist umana

- stato allineato ai fatti
- blocchi e rischi esplicitati

## Errori comuni

- segnare completed senza quality gate
- non registrare decisioni critiche
