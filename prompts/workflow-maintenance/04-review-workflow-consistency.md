# Workflow Maintenance 04 - Review Workflow Consistency

## A cosa serve

Verifica coerenza tra requirement, workflow, prompt, task e documentazione.

## File da leggere

- requirement master
- workflow corrente
- prompts principali
- documentazione hub

## Prompt pronto da copiare

```text
Esegui un consistency review del processo Ready2Agent Learn.
Confronta requirement, workflow, prompt, task e documentazione.
Restituisci:
- gap trovati
- impatto
- fix raccomandati in ordine di priorita
```

## Output atteso

- report di consistenza con azioni correttive

## Checklist umana

- gap concreti, non generici
- fix applicabili subito

## Errori comuni

- audit superficiale
- ignorare link rotti o riferimenti obsoleti
