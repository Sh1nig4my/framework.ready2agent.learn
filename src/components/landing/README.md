# Landing Components (Skeleton)

## Scopo della cartella

Ospita componenti dedicati alla landing pubblica di Ready2Agent Learn.

## Perche esiste nel metodo Ready2Agent

Separare componenti landing aiuta a mantenere la comunicazione divulgativa ordinata e riusabile.

## Cosa andra qui durante il caso studio

- hero section component
- cards per percorsi utente
- sezioni comparative prima/dopo

## Esempi concreti

- `hero.tsx`
- `profile-paths.tsx`

## Cosa non mettere qui

- componenti dashboard privata
- componenti dipendenti da auth enterprise

## Come aiuta un coding agent

Fornisce confini chiari per evolvere la landing senza introdurre codice fuori obiettivo Learn.
