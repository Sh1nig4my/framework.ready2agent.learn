# UI Components (Skeleton)

## Scopo della cartella

Contiene primitive UI leggere riusabili nella versione Learn.

## Perche esiste nel metodo Ready2Agent

Un set minimo di componenti riduce duplicazione e migliora consistenza visiva durante iterazioni guidate da agenti.

## Cosa andra qui durante il caso studio

- button base
- card base
- badge base

## Esempi concreti

- `button.tsx`
- `card.tsx`

## Cosa non mettere qui

- logica business
- dipendenze enterprise

## Come aiuta un coding agent

Permette di riusare primitive stabili invece di creare markup ripetuto e incoerente.
