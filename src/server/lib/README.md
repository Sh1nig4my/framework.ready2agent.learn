# Server Lib (Skeleton)

## Scopo della cartella

Contiene utility tecniche server-side riusabili tra servizi e repository.

## Perche esiste nel metodo Ready2Agent

Centralizzare utility riduce duplicazioni e aiuta a mantenere coerenza tecnica.

## Cosa andra qui durante il caso studio

- helper di validazione
- funzioni comuni di sicurezza
- mapper tecnici

## Esempi concreti

- `validation.ts`
- `security.ts`

## Cosa non mettere qui

- regole di dominio complete
- endpoint API

## Come aiuta un coding agent

Permette all'agente di riusare funzioni comuni invece di ricreare frammenti sparsi nel codice.
